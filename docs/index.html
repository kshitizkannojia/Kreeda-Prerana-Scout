Core Features

Built for PE teachers in rural schools who need a simple, offline tool to track student athlete performance over time.

⏱

Precision Stopwatch

High-accuracy timer using SystemClock.elapsedRealtime() with 10ms intervals for recording sprint times to 2 decimal places.

📋

Batch Entry

Record performance data for an entire class of 30 students in a single session with scroll-safe data persistence.

🏆

Milestone Badges

Automatic "District Level Ready" badge when a student crosses preset sport-specific benchmark values.

📈

Talent Curve

Line graph showing performance improvement over time using MPAndroidChart with cubic Bezier interpolation.

🏅

Leaderboard

Internal school ranking sorted by best performance scores per sport category with gold, silver, bronze highlights.

🔒

100% Offline

Room Database (SQLite) ensures complete functionality without internet — perfect for rural schools with limited connectivity.

Architecture — MVVM Pattern

Clean separation of concerns using Model-View-ViewModel architecture for maintainable, testable code.

View Layer (UI)

XML Layouts, Activities (MainActivity, TrialLoggerActivity, BatchEntryActivity, LeaderboardActivity, TalentCurveActivity), RecyclerView, Material Components

↓ User Actions     ↑ LiveData / State

ViewModel / Controller Layer

Activity Controllers, BenchmarkUtils (13 sport benchmarks), Executors (background threading), Handler (10ms timer callbacks)

↓ CRUD Requests     ↑ Query Results

Model Layer (Data)

AppDatabase (Room), AthleteDao, TrialRecordDao, Athlete Entity, TrialRecord Entity (@ForeignKey CASCADE)

Technology Stack

Built with proven Android technologies for reliability and performance on low-end devices.
